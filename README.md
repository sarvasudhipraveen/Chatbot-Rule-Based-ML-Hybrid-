Chatbot (Rule-Based + ML Hybrid)
Project Overview

This project demonstrates the development of a Hybrid Chatbot using both Rule-Based and Machine Learning approaches. The chatbot can understand user queries, identify intents, and provide suitable responses.

The project uses Natural Language Processing (NLP) techniques such as text preprocessing, TF-IDF vectorization, and intent classification using a Machine Learning model.

Objective
Build a chatbot capable of answering user queries.
Implement rule-based responses for common keywords.
Use Machine Learning for intent prediction.
Gain practical experience in NLP and chatbot development.
